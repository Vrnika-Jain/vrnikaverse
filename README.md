# 🌌 VrnikaVerse

**VrnikaVerse** is an interactive, scroll-driven portfolio designed as a digital universe — where each section represents a different dimension of skills, projects, experience, and AI vision.

🔗 **Live Demo:** [vrnikaverse](https://vrnikaverse.netlify.app/)

---

## ✨ Features

* 🎮 Gamified UI with zones (Origin → Portal Room)
* 🧠 Skill visualization as a dynamic “Data City”
* ⚔️ Project Arena with interactive cards
* 🏆 Certification Vault (horizontal scroll carousel)
* 🧬 Experience timeline with smooth animations
* 🔮 AI Core philosophy section
* 🌀 Portal-based navigation (LinkedIn, GitHub, Email)

---

## 🛠 Tech Stack

* HTML5
* CSS3 (Advanced animations & layouts)
* Vanilla JavaScript
* Canvas API (background rendering)
